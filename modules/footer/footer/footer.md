## 如何使用
* 1、在项目里的Components文件夹里新建一个文件夹Footer，里面存放的文件就是 [组件源码](#组件源码)   的文件结构。

* 2、在项目的模版页面`BasePage`的`index.js`里声明和引用`Footer`页尾组件。

```js
import Footer from '../../Components/Footer';   //声明，路径要填写正确 

<Footer />           //引用

```
